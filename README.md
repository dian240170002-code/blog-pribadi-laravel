## ⚙️ Cara Menjalankan

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/blog-pribadi.git
```

### 2. Masuk ke Folder

```bash
cd blog-pribadi
```

### 3. Install Dependency

```bash
composer install
```

```bash
npm install
```

### 4. Salin File Environment

```bash
cp .env.example .env
```

### 5. Generate Key

```bash
php artisan key:generate
```

### 6. Atur Database

Edit file **.env**

```
DB_DATABASE=blog_pribadi
DB_USERNAME=root
DB_PASSWORD=
```

### 7. Migrasi Database

```bash
php artisan migrate
```

### 8. Jalankan Project

```bash
php artisan serve
```

Buka browser:

```
http://127.0.0.1:8000
```
